# Voice Assistant

A local voice assistant built in Python that listens for user commands and responds using speech recognition and text-to-speech.

This project is designed to run locally due to microphone access and API key requirements.

## Features
- Voice input via microphone
- Speech-to-text processing
- Command parsing
- Spoken responses

## Tech Stack
- Python
- speechrecognition
- pyttsx3 / gTTS (or whatever you used)

## How to Run Locally

### Prerequisites
- Python 3.8+
- A working microphone
